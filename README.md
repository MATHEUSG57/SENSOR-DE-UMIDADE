# SENSOR-DE-UMIDADE
Trabalho entregue ao professor Epaminondas de um sensor para aplicação  na irrigação
<p>&nbsp;</p>
<h1><a id="user-content-t&iacute;tulo-ou-nome-do-projeto" class="anchor" href="https://github.com/Epaminondaslage/Como-montar-seu-README#t%C3%ADtulo-ou-nome-do-projeto" aria-hidden="true"></a>Sensor de umidade para aplica&ccedil;&atilde;o de irriga&ccedil;&atilde;o&nbsp;</h1>
<p>O presente projeto, tem como objetivo apresenta, solu&ccedil;&atilde;o para automa&ccedil;&atilde;o da irri&ccedil;&atilde;o residencial e comercial</p>
<h1><a id="user-content-descri&ccedil;&atilde;o" class="anchor" href="https://github.com/Epaminondaslage/Como-montar-seu-README#descri%C3%A7%C3%A3o" aria-hidden="true"></a>Descri&ccedil;&atilde;o</h1>
<p>Com objetivo de reduzir os gastos com a irriga&ccedil;&atilde;o. Esse sitema, ajudano no manejo correto da &Aacute;gua, tanto em estufas quanto em plant&ccedil;&otilde;es convencionais. Isso possibilita maior efeci&ecirc;ncia e ec&ocirc;nomia.</p>
<p>&nbsp;</p>
<h1><a id="user-content-pr&eacute;-requisitos" class="anchor" href="https://github.com/Epaminondaslage/Como-montar-seu-README#pr%C3%A9-requisitos" aria-hidden="true"></a>Pr&eacute;-requisitos</h1>
<p>Esse projeto foi desnvolvido na Ide arduinio, para ser aplicado nos diversos arduino.</p>
<h1><a id="user-content-instala&ccedil;&atilde;o" class="anchor" href="https://github.com/Epaminondaslage/Como-montar-seu-README#instala%C3%A7%C3%A3o" aria-hidden="true"></a>Instala&ccedil;&atilde;o</h1>

<h1><a id="user-content-uso" class="anchor" href="https://github.com/Epaminondaslage/Como-montar-seu-README#uso" aria-hidden="true"></a>Uso</h1>
<p>Esse projeto, tem indicadores de taxa de humidade e acionameto da bomba/sol&eacute;noide.</p>
<p>Pode usar tanto uma sol&eacute;noide tanto uma bamba em uma saida r&eacute;le para ativar a irriga&ccedil;&atilde;o.</p>
<p>Esse projeto tamb&eacute;m pode-se controla a irriga&ccedil;&atilde;o pelo celular</p>
<p>A entrada do sensor de umidade &eacute; digital, com seu renge regulado no proprio sensor.&nbsp;</p>
